## Database Schema

### Engineer Assignments Table
```sql
CREATE TABLE engineer_org_memberships (
    engineer_id UUID REFERENCES engineers(id),
    org_id UUID REFERENCES organizations(id),
    role TEXT,
    created_at TIMESTAMP DEFAULT NOW(),
    PRIMARY KEY (engineer_id, org_id)
);
```

### Query to Get Engineers for a Manufacturer
```sql
SELECT 
    e.id, e.name, e.skills, e.email, e.phone,
    m.role
FROM engineer_org_memberships m
JOIN engineers e ON e.id = m.engineer_id
WHERE m.org_id = 'f1c1ebfb-57fd-4307-93db-2f72e9d004ad'  -- Philips
ORDER BY e.name;
```

### Query to Get Engineer Count
```sql
SELECT COUNT(DISTINCT engineer_id) 
FROM engineer_org_memberships 
WHERE org_id = 'f1c1ebfb-57fd-4307-93db-2f72e9d004ad';
```
